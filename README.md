1. Clone the repository:

2. Navigate to the project folder:
```bash

```

3. Install frontend dependencies:
```bash
cd frontend
npm install
```

4. Install backend dependencies:
```bash
cd ../backend
npm install
```

5. Start the backend:
```bash
npm run dev
```

6. In a new terminal tab, go to the frontend folder and start the dev server:
```bash
cd frontend
npm run dev
```
